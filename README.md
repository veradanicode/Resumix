# 📝 ResumeMix

**ResumeMix** is a simple, customizable resume builder built with **React** and **Node.js**. It lets users easily fill out their details, choose from beautiful templates, and download a professional resume as a **PDF** — all in one session, no sign-up required.

---

## 🚀 Features

- 🧾 Easy-to-use resume form  
- 🎨 Customizable templates  
- ⚡ Real-time resume preview  
- 📄 Download as PDF (one-click)  
- 🔐 No account or login required

## 🛠 Tech Stack

- **Frontend**: React  
- **Backend**: Node.js, Express  
- **PDF Generation**: `react-to-print` (or `jspdf` if you choose)  
- **Styling**: CSS3

## 🧑‍💻 How to Run Locally

### 1. Clone the Repository

```bash
git clone https://github.com/veradanicode/resumemix.git
cd resumemix

2. Install Dependencies
For the frontend:

cd client
npm install
npm start

For the backend:

cd server
npm install
npm run dev

Make sure both the frontend and backend are running on different ports (e.g., frontend on 5173, backend on 5000).
